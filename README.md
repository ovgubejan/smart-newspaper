#Smart Personnel Newspaper (SPN)
> **An AI-driven content curation engine transforming news streams into personalized, print-ready digital newspapers.**



## Project Overview
SPN is an automated system designed to crawl, classify, and deliver hyper-personalized news. Built as part of the **SENG448 Engineering Design** course, the project follows a strict Waterfall methodology with a focus on UML-driven architecture.

### Current Status
**Phase:** 1. Requirements Analysis  
**Progress:** 15%  
**Next Milestone:** Software Design Document (SSDD) - Week 3
